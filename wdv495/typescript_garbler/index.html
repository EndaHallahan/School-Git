Text Garbler

Goal: create an effect that looks like a piece of text is being rapidly deciphered from an "encrypted" (garbled) string. Make it interactable through click or hover.

Attempt number three eschews random indexes in favor of a random order. It creates an array 0..n-1 where n is the length of the input string. This array is then shuffled using the Knuth algorithm. On each animation frame, the text is re-garbled, and then the last element in the array is removed and the corresponding index in the string is fixed. This removes the need for a maximum time duration.
